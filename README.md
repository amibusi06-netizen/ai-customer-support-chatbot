# AI Customer Support Chatbot

## 🇫🇷 Résumé

Ce projet présente la création d’un agent IA capable de répondre automatiquement aux questions fréquentes des utilisateurs dans un contexte de service client.

Le chatbot utilise un système de prompt structuré afin de fournir des réponses claires, rapides et pertinentes.

---

## 🇬🇧 Project Overview

This project demonstrates the creation of an AI chatbot designed to automatically respond to user questions in a customer support context.

The chatbot uses structured prompting to deliver clear, fast, and relevant answers.

---

## Outils utilisés / Tools Used

* Voiceflow
* Modèle d’intelligence artificielle (LLM)
* Prompt engineering
* ChatGPT (assistance for structuring prompts and workflow)

---

## Fonctionnement / How it works

### 🇫🇷

1. L'utilisateur pose une question
2. L’IA interprète la demande
3. Le système utilise des informations prédéfinies
4. Une réponse claire et adaptée est générée

### 🇬🇧

1. The user asks a question
2. The AI interprets the request
3. The system uses predefined information
4. A relevant answer is generated

---

## Résultats / Results

* Réponses automatiques aux utilisateurs
* Amélioration de l’expérience utilisateur
* Système rapide et efficace

---

## Compétences démontrées / Skills Demonstrated

* Intégration d’IA
* Prompt engineering
* Conception de chatbot
* Logique produit / UX

---

## Contexte / Context

### 🇫🇷

Ce projet utilise un centre de formation fictif inspiré d’Alegria comme cas d’usage, afin de simuler un environnement réel de service client.

### 🇬🇧

This project is based on a fictional training center inspired by Alegria, used as a real-world scenario to simulate customer support interactions.

## Note

### 🇫🇷 Ce projet a été réalisé avec l’aide de ChatGPT pour structurer le prompt et organiser le workflow, tout en comprenant et en implémentant chaque étape de manière autonome.

### 🇬🇧 This project was carried out with the help of ChatGPT to structure the prompt and organize the workflow, while understanding and implementing each step autonomously.
